# Сравнительный анализ браузеров: Firefox vs Brave

## 1. Введение

Данный отчет представляет результаты сравнительного анализа браузеров Firefox и Brave по ряду ключевых критериев. Целью является предоставление детальной информации для разработчиков и технических специалистов для оценки производительности, безопасности и конфиденциальности.

## 2. Методология

Тестирование проводилось на компьютере со следующими характеристиками:

*   **Операционная система:** Windows 10 Pro (версия 22H2)
*   **Процессор:** Intel Core i7-8700K
*   **Оперативная память:** 16 GB DDR4
*   **Видеокарта:** NVIDIA GeForce GTX 1070

Версии браузеров:

*   Firefox: 123.0.1 (64-bit)
*   Brave: 1.63.165 Chromium: 122.0.6261.94 (Official Build) (64-bit)

Использовались следующие инструменты:

*   **Task Manager (Windows):** Для измерения потребления RAM и CPU.
*   **HTML5Test.com:** Для оценки совместимости с HTML5.
*   **WebPageTest.org:** Для измерения скорости загрузки страниц (First Contentful Paint).
*   **BrowserAudit.com:** Для оценки параметров конфиденциальности.

**Веб-сайты для теста скорости загрузки:**

*   Google.com
*   YouTube.com
*   Wikipedia.org
*   Amazon.com
*   Reddit.com
